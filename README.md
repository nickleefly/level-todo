# level-todo

A todo list using leveldb

# example

To add a new list,

`todo add "this is a name" "this is a value"` or `todo add read https://www.npmjs.com`

# Usage

```
usage:

  todo add name value
  Add a new list.

  todo get name
  Get a list

  todo rm/done name
  Mark a list as done and remove it by the given name.

  todo list
  Show all the todo list

  todo destroy
  Clear all the list.
```
